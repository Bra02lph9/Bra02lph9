## 🚀 Featured Projects

### 🧬 Sex-Specific GWAS Analysis Pipeline

A modular bioinformatics pipeline designed to compare male-specific, female-specific, and shared genetic signals in complex diseases.

**Main components:**

* GWAS preprocessing and quality filtering
* Significant SNP and locus comparison
* FUMA and MAGMA integration
* GTEx eQTL validation
* Bayesian colocalization
* Linkage disequilibrium validation
* Pathway enrichment
* Protein interaction networks
* Candidate gene prioritization

### 🧠 Brain Tumor Radiogenomic Classification

A medical imaging project for predicting the **MGMT promoter methylation status** from multimodal brain MRI scans.

**Main components:**

* DICOM image preprocessing
* FLAIR, T1wCE, T1w, and T2w modalities
* 3D convolutional neural networks
* Transfer learning with pretrained 2D models
* Cross-validation and performance evaluation
* ROC curves, confusion matrices, and classification reports
